### GeoHackWeek project: GEE images in Folium
For his project, [Ryan McCarthy](https://github.com/mccarthyryanc) figured out how to use the GEE Python API to render GEE Tile Layers in a [Folium](https://github.com/python-visualization/folium) interactive map. The code, including a [Jupyter notebook example](http://nbviewer.jupyter.org/github/mccarthyryanc/folium_gee/blob/master/TilesExample.ipynb), is on [Ryan's github repository.](https://github.com/mccarthyryanc/folium_gee) *Note that in the notebook the GEE image will no longer show up, because GEE caches these images for only a couple of days or so.*
